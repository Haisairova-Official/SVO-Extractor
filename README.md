# TeaGFX Extractor
适用于茶引擎的SVO+AFB私有格式的DDS提取器
### *新附属项目：[TeaGFX Replacer](https://github.com/Haisairova-Official/TeaGFX-Replacer/)*

**TeaGFXExtractor** 是一款专为 **Chunithm**以及**Maimai(旧框体)** 及其相关街机平台设计的资源提取工具。本工具通过深度扫描 `.svo`与`.afb` 容器文件，能够全自动、批量化地提取出内部封装的 **DDS 贴图资源**。
对于AFB文件，感谢文档指引，我们现在已经可以直接提取其真实文件名。

---

## 🛠️ 核心功能

* **智能特征扫描**：基于 `DDS ` 头部特征码（十六进制：`44 44 53 20`）进行毫秒级定位截取。
* **全自动目录遍历**：支持递归扫描选定目录及其所有子文件夹下的 `.svo` 文件，无需手动逐个文件夹操作。
* **结构化导出**：自动为每个 `.svo` 文件创建独立的存放目录，确保提取出的数千张贴图井然有序。
* **实时进度监控**：可视化进度条与详细的运行日志，处理状态一目了然。
* **高性能提取**：优化了二进制文件流读取逻辑，支持处理 GB 级别的超大 SVO 镜像。

## 🚀 快速开始

### 使用步骤
1.  **路径设置**：
    * **扫描目录 (Path)**：点击 `BROWSE` 选择包含 `.svo,.afb` 文件的文件夹（**允许**子文件夹，通常在游戏目录的 `data` 下）。
    * **输出目录 (Export Path)**：选择你想存放提取出的贴图的位置。
2.  **启动任务**：点击底部的 `START EXTRACTION` 按钮。
3.  **查看结果**：程序执行完成后，会自动在输出目录生成分类文件夹，内含所有提取出的 `.dds` 文件。

## 📂 技术背景

根据 [GekiChuMai Localized Document](https://gzlxz190614.github.io/GekiChuMaiLocalizedDocument/Chunithm/Extra/2.svo.html) 的技术文档描述：

1.  **容器结构**：本质上是一个不带文件索引表的原始数据包，用于存储游戏资源。
2.  **提取逻辑**：由于文件内部通常将多个 DDS 文件直接拼接，程序通过搜索字符串 "DDS " 作为文件起始标志，并截取至下一个标志位出现之前的内容。
